## my-app-go

Это простое CRUD-приложение на Go с использованием базы данных MySQL. Веб-интерфейс позволяет управлять следующими сущностями:

- Преподаватели (Teachers)
- Студенты (Students)
- Курсы (Courses)
- Записи на курсы (StudentsCourses)

## Зависимости

- Go 1.20 или новее
- MySQL (или совместимая СУБД)

## Функциональность

Каждый модуль (teachers, students, и т. д.) поддерживает:

- Просмотр списка записей
- Создание новой записи
- Редактирование записи
- Удаление записи

## 📁 Шаблоны

Все HTML-шаблоны хранятся в папке `templates/`. Названия файлов следуют формату:

<модуль>_<действие>.html

Примеры:

- `teachers_index.html`
- `students_create.html`
- `courses_edit.html`
- `students_courses_index.html`
